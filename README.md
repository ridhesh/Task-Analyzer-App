# Smart Task Analyzer

A intelligent task management system that scores and prioritizes tasks based on multiple factors including urgency, importance, effort, and dependencies.

## Features

- **Smart Priority Scoring**: Algorithm that considers multiple factors to calculate task priority
- **Multiple Sorting Strategies**: Choose between different prioritization approaches
- **Task Analysis**: Get detailed insights into why tasks are prioritized
- **Top Suggestions**: Receive personalized recommendations for what to work on next
- **Bulk Task Import**: Load tasks via JSON for quick setup
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: Your tasks are saved automatically
- **Django REST API**: Robust backend with full CRUD operations
